# Palindrome Checker

## Описание программы:

Эта программа предназначена для проверки, является ли введенное слово палиндромом. Палиндром - это слово, которое читается одинаково как слева направо, так и справа налево. Программа игнорирует регистр букв и пробелы, чтобы корректно определить палиндромы.

## Описание работы:

Программа использует следующие шаги для определения, является ли слово палиндромом:

1. Удаляет все пробелы из слова.
2. Преобразует все буквы в слове в нижний регистр.
3. Использует класс `StringBuilder` для эффективного переворачивания слова.
4. Сравнивает оригинальное слово с его перевернутым вариантом.

### Использованные методы:

- `isPalindrome(String word)`: Основной метод, который принимает слово в виде строки и возвращает `true`, если оно является палиндромом, и `false` - в противном случае.
- `replaceAll("\\s", "")`: Удаляет все пробелы из слова.
- `toLowerCase()`: Преобразует все буквы в слове в нижний регистр.
- `reverse()`: Переворачивает строку с помощью `StringBuilder`.

### Применение "Five Why's" принципа:

1. **Почему программа удаляет пробелы?**
   - Чтобы палиндром можно было проверить правильно, игнорируя пробелы.

2. **Почему все буквы преобразуются в нижний регистр?**
   - Это делает программу более гибкой, поскольку регистр букв не влияет на палиндром.

3. **Почему используется `StringBuilder` для переворачивания строки?**
   - Это позволяет эффективно сравнивать оригинальное слово и его перевернутую версию.

4. **Почему программа возвращает `true` для палиндрома и `false` для не палиндрома?**
   - Это стандартный способ сообщения пользователю о результате проверки.

## О тестах:

Тестирование функциональности программы включает в себя следующие сценарии:

- Проверка правильного определения палиндрома для слов, таких как "level".
- Проверка игнорирования регистра букв для ввода типа "Able was I saw eLba".
- Проверка определения не палиндрома для слов, таких как "hello".
- Проверка работы программы с фразами, содержащими пробелы, например, "A man a plan a canal Panama".
